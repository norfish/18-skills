# AI Skills 仓库

这是一个用于管理和组织 AI 技能的仓库。

## 目录结构

```
.
├── README.md          # 本文件
├── skills/           # 所有技能目录
│   ├── README.md     # 技能索引
│   └── example/      # 示例技能
│       ├── README.md
│       └── SKILL.md
└── .gitignore
```

## 技能格式

每个技能应该包含：
- `README.md`: 技能说明和使用文档
- `SKILL.md`: 技能详细指令（如果有）
- 其他相关文件

## 如何使用

1. 在 `skills/` 目录下创建新目录
2. 按照示例格式编写你的技能文件
3. 更新 `skills/README.md` 索引

## 添加新技能

参考 `skills/example/` 目录中的示例结构。
